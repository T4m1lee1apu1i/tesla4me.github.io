Auch Informatikerinnen und Informatiker der Fachrichtung Plattformentwicklung spielen gerne mal ein kleines Game zur Auflockerung. Nun ist aber ist ihre Leidenschaft die Programmierung von Automatisierungen. Warum also nicht einen Bot programmieren der mir die Arbeit abnimmt? Aber was ist ein Bot?

Ein Bot ist eine wiederholende Automatisierung bei der keine weitere Interaktion nötig ist. Der Bot arbeitet für einen Benutzer.

Um einen Bot zu programmieren braucht es die Programmiersprache Python.

Python ist bereits auf unseren Geräten installiert. Wenn du zu Hause Python nutzen willst kannst du das Programm gratis bei www.python.org beziehen. Nach dem Download startest du die Installationsdatei und wählst «Install Now» Wichtig! Vorher die Checkbox “add Python to path” (!) wählen.



Danach musst du noch Programmbibliotheken installieren. Diese Bibliotheken vereinfachen die Programmierung. Um sie zu aktivieren öffnest du die sogenannte Kommandozeile mittels Befehl CMD. Gib im schwarzen Konsolenfenster nacheinander die 6 Befehle ein:

pip3 install pywin32

pip3 install keyboard

pip3 install pyautogui

pip3 install opencv-python

pip3 install pillow

Danach kannst du das Konsolenfenster schliessen.
