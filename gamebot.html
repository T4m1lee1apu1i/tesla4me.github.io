Was ist ein Bot?

Eine wiederholende Automatisierung bei der keine Interaktion nötig ist. Der Bot arbeitet für einen Benutzer.

Was brauche ich um den Bot zu programmieren?



1. Python installieren:

Download von www.python.org dann nach dem Start der Installation «Install Now» wählen und die Checkbox “add Python to path” (!) wählen.



2. Bibliotheken installieren:

Konsole mittels Befehl CMD öffnen. Im schwarzen Konsolenfenster nacheinander die 6 Befehle eingeben:



pip3 install pywin32

pip3 install keyboard

pip3 install pyautogui

pip3 install opencv-python

pip3 install pillow

Funktion erstellen: «gehe zur Position X Y» “drücke Maus links” «Maus links loslassen»:
