Was ist ein Bot? -> eine wiederholende Automatisierung bei der keine Interaktion nötig ist. Der Bot arbeitet für einen Benutzer. In den nachfolgenden Abschnitten in dieser Spalte ist die Struktur der Webseite beschrieben. Diese Überschrift findest du im HTML-Code unter "section".

Damit kannst du zum Beispiel eine Einleitung oder Beschreibung der nachfolgenden Felder schreiben.
Was brauche ich um den Bot zu programmieren?

1. Python installieren: Download von www.python.org dann nach dem Start der Installation «Install Now» wählen und die Checkbox “add Python to path” (!) wählen.

2. Bibliotheken installieren: Konsole mittels Befehl CMD öffnen. Im schwarzen Konsolenfenster nacheinander die 6 Befehle eingeben:

pip3 install pywin32

pip3 install keyboard

pip3 install pyautogui

pip3 install opencv-python

pip3 install pillow
